## 🚀 Overview  

Gradify is an **AI-driven career guidance platform** that helps students discover the right career path based on their skills, interests, and academic background.  

Using **Machine learning + LLM APIs**, Gradify provides:  

- career suggestions  
- skill analysis  
- recommended learning paths  
- motivational insights  
- job future insights  

---

## ✨ What It Does  

✔ Analyse user skills  
✔ Evaluate interest  
✔ Suggest career path  
✔ Recommend courses  
✔ Shows real-world growth opportunities
